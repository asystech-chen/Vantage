# Vantage Browser 更新日志

## v150.0-1 — 2026-04-30

- **升级至 Firefox 150.0**，同步 LibreWolf 150.0-1 上游安全更新和补丁。

- **新增隐私仪表盘（about:protections）**，重新设计保护状态页面，以统计卡片展示已拦截的跟踪器、Cookie、指纹脚本、加密货币挖矿与社交跟踪等数据，新增保护状态面板（绿色/红色指示器），支持中英文。

- **新增 Windows 安装程序 Publisher 名称**，现在显示为 Vantage 而非 Mozilla。

- **新增 Windows 安装器中文化**，安装程序界面现在会优先显示简体中文，并在卸载时询问是否删除配置文件。

- **更新检查改进**：移除 24 小时间隔限制，改为每次启动时检查更新，并新增自定义更新服务器地址支持。

- **修复设置面板复选框同步问题**：IPv6 开关、WebGL 开关、同步开关等单选项现在可以正确保存配置到 about:config。

- **恢复 about:debugging 按钮**到设置面板。

- **移除 Tampermonkey 默认安装**。

- **截图工具等扩展默认固定到工具栏**。

- **新增默认图标尺寸**：22×22、24×24、256×256 像素图标。

- **新增 Linux ARM64 打包支持**，包括 deb、rpm、AppImage、portable tar.gz。

- **更新 .gitignore**，排除 .exe、.dmg 等构建产物。

- **安全增强**（来自 LibreWolf 150.0-1）：
  - 防止通过 branding 进行浏览器指纹识别
  - HTTPS 自动升级增强，自动升级本地连接
  - DoH 查询改为 POST 方法（隐私更好）
  - 禁用 OCSP（减少隐私泄露，已有 CRLite 补偿）
  - 禁用企业根证书（防止中间人攻击）
  - 禁用 HTTP3 0-RTT（防止重放攻击）
  - 禁用弱加密套件

## v149.0-2 — 2026-04-03

- **新增 AI 侧边栏功能**（默认关闭），集成 DeepSeek、通义千问、豆包、ChatGPT、Gemini、Claude 等 AI 服务，可在设置中一键开启。

- **新增隐私仪表盘（about:protections）**，重新设计保护状态页面，支持中英文，隐私设置页新增快捷入口按钮。

- **支持分屏浏览（Split View）**，右键标签页即可使用，继承自 Firefox 149 原生功能。

- **新增 macOS x86_64 交叉编译支持**，首次成功产出 DMG 安装包，可通过 Rosetta 2 在 Apple Silicon 上运行。

- **新增 macOS ARM64 及 Linux ARM64 编译配置**（mozconfig）。

- **新增卸载器改进**：移除 Mozilla 卸载调查问卷，卸载完成后询问是否清理用户数据。

- **新增安装器中文化补丁**，优先加载中文 locale 资源。

- **打包系统新增多架构支持**。

- **RPM 打包启用多线程压缩**（xzmt），提升打包速度。

- **更新 .gitignore**，排除 .deb/.rpm/.AppImage 等构建产物，清理 git 历史中误提交的大文件（仓库体积从 499MB 缩减至 12MB）。

- **DMG 打包修复**：自动检测并设置 Mach-O 可执行文件权限，解决安装后无法运行的问题。

- **macOS SDK 版本要求放宽至 26.1**（原 26.2）。

- **Windows 安装程序显示语言改为简体中文**，用户现在可以在卸载时决定是否删除配置文件了。

## v149.0-1 — 2026-03-31

- **配置文件路径迁移**：Vantage 配置文件不再与 LibreWolf 共享目录，Windows 下为 `%APPDATA%\Vantage\`，Linux 下为 `~/.vantage/`，macOS 下为 `~/Library/Application Support/Vantage/`，并提供配置文件迁移工具。

- **新增自动更新检查**，后台每 24 小时检查一次更新，支持忽略特定版本，检查到更新后将提醒。

- **设置面板新增"更新"区块**，可手动开启/关闭更新检查，「关于」页面的手动检查更新依然保留。

- **恢复"设为默认浏览器"选项**，可在浏览器设置中查看和设置。

- **修复设置中的配置项说明链接**，现在这些链接指向 Mozilla 官方说明页面。

- **修复 Windows 主题加载问题**。

- **补全 WebGL 权限相关翻译**（简繁体中文）。

- **繁体中文已重新启用并可用**。

- **移除打包目标对 package 的冗余依赖**，加速重复打包流程。

## v149.0-0 — 2026-03-28

- **合并上游补丁**，更新版本号到 149.0。

- **修复增强型跟踪保护的 Bug**，现在将默认启用严格保护模式。

- **现在开始默认启用 DRM**，以兼容部分需要版权保护的视频网站。

- **调节了一些其他配置**，增强易用性。

- **中国大陆用户现在应该无需额外下载 uBlock Origin 插件**，插件将自动安装到浏览器中。

## v149.0-0 — 2026-03-15

- **合并上游补丁**。

- **增加更新检查功能**，由用户自己决定是否更新。

## v149.0-0 — 2026-03-06

- **初始 Release**，支持 Windows 和 Linux。
